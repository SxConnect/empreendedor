Documentação do Painel

O que é o SxConnect Dashboard

É uma interface unificada para operar atendimento, automação, CRM, IA e integrações. O objetivo é reduzir tarefas repetitivas, centralizar informações e aumentar a previsibilidade da operação.

Primeiros passos após instalar

  1. Abra o painel em http://127.0.0.1:9000.
  2. Confirme se o status geral está ativo.
  3. No menu lateral, abra Documentação para seguir este guia.
  4. Antes de usar módulos avançados, revise conexões em Início e em Assistente.
  5. Configure o WhatsApp, se necessário, e depois vá para Produtos para alimentar o assistente com base de conhecimento.

Módulos principais

  • Início: visão rápida do ecossistema conectado e status dos serviços.
  • Assistente: chat em linguagem natural que pode usar agentes/contextos para responder operacionalmente.
  • Memória: armazenamento de anexos/referências usados pelo fluxo.
  • Produtos: cadastro de itens que ajudam a IA a responder com mais precisão.
  • Relatórios: área para revisar documentos/relatórios e autorizar quando necessário.
  • WhatsApp: painel integrado para contatos e conversas.
  • Kambam: funil de vendas com colunas configuráveis.
  • Agendamento: gestão de horários e disponibilidade.
  • Documentação: guias e referências rápidas.

Checklist de uso recomendado

  • Verificar status dos serviços em Início.
  • Validar se o WhatsApp está conectado.
  • Cadastrar produtos/serviços em Produtos.
  • Configurar funil em Kambam.
  • Ajustar regras de Agendamento conforme horário de funcionamento.
  • Usar Assistente para testar respostas e fluxos.

Configurações comuns

  • Painel principal: 9000.
  • Integração com WhatsApp: 8080.
  • Contatos/serviços auxiliares: 9002.
  • Dashboard nativo Hermes: 9119.

Solução rápida

  • Se a página carregar em branco, force uma atualização.
  • Veja o console do navegador para erros de JS/CORS.
  • Valide se os serviços auxiliares estão atendendo nas respectivas portas.
