Visão geral

Essa página reúne as instruções para operar e avaliar os agentes de IA do SxConnect, incluindo uso no assistente, configuração e verificação rápida.

Passos básicos

  1. Abra Assistente no menu para interagir com o agente principal.
  2. Use Documentação para consultar capacidades e fluxos.
  3. Valide respostas em cenários reais antes de expor para clientes.
  4. Ajuste contexto/base de conhecimento quando necessário.

Boas práticas

  • Prefira respostas curtas e objetivas para atendimento.
  • Revise saídas antes de usar como abordagem comercial.
  • Mantenha conexões verificadas antes de fluxos automáticos.

Status esperado

  • Assistente respondendo sem erros.
  • Conexões ativas no painel.
  • Dados atualizados em Início.
